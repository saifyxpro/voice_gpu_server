# Chatterbox Turbo — tag reference

Complete list (Turbo only). Use **one at a time** in production unless the skill budget allows two on a long script.

| Tag | Sound | Use when | Skip when |
|-----|-------|----------|-----------|
| `[clear throat]` | Throat clear | Opening a call, before a formal announcement | Mid-conversation, repeated use |
| `[sigh]` | Sigh | Empathy, acknowledging hassle, softening news | Happy or excited lines |
| `[shush]` | Quiet “shh” | Confidential aside, calming | Standard business flow |
| `[cough]` | Cough | Rare realism beat | Professional polish needed |
| `[groan]` | Groan | Mild frustration (sparingly) | Client-facing empathy moments |
| `[sniff]` | Sniff | Emotional hesitation (rare) | Most outbound scripts |
| `[gasp]` | Gasp | Genuine surprise | Routine reminders |
| `[chuckle]` | Light laugh | Warm rapport, light humour | **Default go-to** — still max 1/turn |
| `[laugh]` | Fuller laugh | Clear joke or punchline | Compliance / filing calls |

## Chatterbox vs ElevenLabs v3

| Chatterbox | ElevenLabs (not this server) |
|------------|------------------------------|
| `[clear throat]` | `[clears throat]` |
| `[chuckle]` | `[chuckles]` |
| `[laugh]` | `[laughs]` |
| `[sigh]` | `[sighs]` |
| — | `[curious]`, `[excited]`, `[whispers]` |

Eleven delivery/emotion tags are **not** supported by Chatterbox Turbo.
